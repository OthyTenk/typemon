# About TypeMon
TypeMon is a multiplayer competitive typing game built with Next.js, TypeScript, TailwindCSS, Prisma, and MongoDB. Players can create or join game lobbies with up to 4 participants and compete in real-time typing challenges across multiple languages.

Key Features
- Multiplayer Gaming: Real-time competitive typing matches via Socket.IO
- Multi-language Support: Practice typing in different languages
- Performance Tracking: Tracks WPM, CPM, accuracy, and typing history
- User Authentication: Secure login with email/password and OAuth providers
- Live Statistics: Real-time player progress visualization during matches

## Tech stack

- Next.js (App router)
- React
- TailwindCSS
- Typescript
- Zustand
- Prisma
- MongoDB
- Socket.IO
- Next-Auth
